DevToolChest.com provides direct homepage, CDN and download links to popular frameworks and libraries such as NodeJS, Knockout.js and jQuery. The front end is built on AngularJS MVC for JSON data fetching, templating and live search (application consists of 1 router, 2 controllers, 1 filter, 1 factory and 2 views) along with Twitter Bootstrap and AngularStrap. The site is built with Ruby on Rails and deployed using Heroku.
